其它要做的事

  • 虚拟机的清理、存档
  • Zotero 的处理，论文库的存档与复原（使用文章记录）
  • 课题与学业文件夹的合并
  • Github 仓库的整理
  • 论文笔记的整理（图片重命名和标题问题）
  • 科研记录的整理（不同代码版本间的关系）
  • 毕业相关的流程（三方、报到证、图书馆、校园卡、退宿）
  • 学生期间相关账户的注销
  • 取关学生阶段的一些公众号，卸载相关的APP
  • 相关纸质文档的留存，包括成绩单、就读证明等，之后可能有用
  • 毕业照：单人、同宿舍、全专业的合照（学位服），对学校各个地方拍照留存
  • 物品整理、打包、投递
  • 医保卡的注销问题
  • 如果学校开放，去昌黎玩一趟
  • 策划毕业旅行
